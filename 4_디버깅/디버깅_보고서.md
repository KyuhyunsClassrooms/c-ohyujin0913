## 🐛 발견한 오류 #1

### 1) 어떤 문제가 발생했나요?
평균 평점을 계산하는 기능(avg)을 실행하려고 하니까
**컴파일 오류(error: expected ‘;’ before ‘}’)**가 발생해 프로그램이 시작되지 않음

### 2) 오류가 발생한 코드
```c
printf("Avg rate: %.2f\n",sum/num)
```

### 3) 왜 오류가 생겼나요?
printf 함수 끝에 세미콜론(;)을 빼먹어서
C 컴파일러가 문장이 끝난 것으로 인식하지 못해서 오류가 발생했습니다.
 
### 4) 어떻게 고쳤나요?
```c
printf("Avg rate: %.2f\n", sum/num);

```

### 5) 고친 후 결과
- [✅ ] ✅ 잘 작동해요!
- [ ] ❌ 아직 문제가 있어요

## 💭 디버깅하면서 배운 점

- 세미콜론(;)을 하나만 빼먹어도 컴파일이 되지 않는다는 걸 배움
- 문자열 비교는 ==이 아니라 strcmp를 사용해야 한다는 것을 알게 됨
- 출력이 복잡하면 한 줄씩 나눠서 적는 것이 더 보기 좋다는 것도 느낌
