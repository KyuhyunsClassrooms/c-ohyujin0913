# 1. 프로그램 설계서 (5점)

## 1. 프로젝트 주제
영상 시청 기록 및 평점 관리 프로그램
## 2. 핵심 기능 목록
(구현할 기능들을 순서대로 나열하세요.)
1.  영상 등록 기능, 사용자가 영상의 제목, 장르, 길이, 날짜, 평점을 입력함
2.  영상 목록 출력 기능, 지금까지 등록된 모든 영상의 정보를 한눈에 볼 수 있도록 출력한다
3.  영상 검색 기능, 입력한 제목에 해당되는 영상을 찾아 정보와 평점을 보여준ㄷ다
4.  평균 평점 계산 기능, 전체 영상의 평점을 합산하여 평균을 계산한다

## 3. 데이터 구조 설계
(프로그램에 필요한 핵심 변수, 배열을 설계하세요. 필요하면 구조체(struct)를 사용할 수도 있습니다.)

**방법 1: 배열 사용 (기본)**
```c
char titles[100][30];
char genres[100][20];
int length[100];
char uploadDate[100][12];
floar rating[100];
int count = 0; // 현재 영상 수 
```

**방법 2: 구조체 사용 (선택)**
```c
struct video {
    char title[30]
    char genre[20];
    int length; //영상 길이(분)
    char uploadDate[12]; //평점(1~5)
    float rating;
};
struct video videos[100];
int count=0;
```
